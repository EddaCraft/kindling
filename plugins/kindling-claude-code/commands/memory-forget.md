Forget (redact) a specific observation. This marks it as redacted and removes it from search results while preserving referential integrity.

Run this command:

```bash
node "${CLAUDE_PLUGIN_ROOT}/scripts/memory-forget.js" $ARGUMENTS
```

Show the result to the user.
